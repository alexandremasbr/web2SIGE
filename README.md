# Web2SIGE

Sistema Integrado de Gestão Empresarial feito com o framework [web2py](http://http://web2py.com/)

Projeto open-source desenvolvido em Python 3 no GNU/Linux.


## Implementações

- Cadastro de produtos, clientes e fornecedores
- Login/Logout
- Módulo de vendas
- Módulo de compras
- Módulo finaceiro
